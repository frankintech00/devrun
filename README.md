#DevRun

This is a [Create React App](https://create-react-app.dev/) (CRA) built for development and learning purposes.

It was initially built by following the React e-commerce course by [John Smigla](https://github.com/john-smilga/react-course-comfy-sloth-starter) and customised to include many new additional features including a new custom API, Mega Menu and others.

## Installation

To install and run the App, use the following commands:

```
npm install
npm start
```

## Building

To build the App, use the following command:

```
npm run build
```

## Hosting

The App is hosted on [Netlify](https://www.netlify.com/).

The App is deployed at https://examclock-fps.netlify.app/.
